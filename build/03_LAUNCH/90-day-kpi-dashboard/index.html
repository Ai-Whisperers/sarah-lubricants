Forecasting vs actuals

At end of each month: 1. Calculate actual revenue vs forecast (from cash-flow-model-12mo.md) 2. If actual <80% of forecast → trigger Plan B (Conservative scenario) 3. If actual >120% of forecast → trigger Plan C (Stretch scenario) 4. Adjust all future months' forecast based on trajectory

Variance alerts

When any KPI deviates by >30% from target for 2 consecutive weeks:

Signal Action
Orders down 30%+ Check: stock, IG engagement, WA response time, competitor launch
AOV down 30%+ Check: bundle pricing, bundle visibility, mix shift
Returns up 30%+ Check: product quality, shipping damage, customer expectations
New followers down 30%+ Check: IG content quality, hashtag strategy, posting cadence
Cash burn > Gs 1M/mo Check: model旅行者 viability, despachante costs, marketing spend
