Kinky Store — Brand & Trademark Assessment

Status: Pre-decision assessment, pending DINAPI verification from Sarah File location: This doc GitHub: https://github.com/Ai-Whisperers/sarah-lubricants/blob/main/01_RESEARCH/brand-identity/kinky-store-assessment.md Date: June 20, 2026

TL;DR

"Kinky Store" is a strong, memorable, polarizing name. Sarah says it's already registered at DINAPI. Three risks to verify before locking:

  1. DINAPI registration belongs to Sarah (not a third party using the same name in PY).
  2. Class coverage includes the categories she actually sells (Class 3 = cosmetics, Class 5 = pharmaceuticals/lubricants, Class 35 = retail). If only Class 35 is registered, she'll need to add 3 + 5 to legally sell under the mark.
  3. The word "Kinky" triggers Meta/Google ad filters aggressively. Paid acquisition will be hard. Organic only at first.

Recommendation: Verify DINAPI first, then lock name. Have 2 alternatives ready in case (Kinklab, Kinkhy).

1. The name — pros and cons

Pros

  • Memorable — short, distinct, easy to type
  • Tone-aligned — "juguetón/divertido" (Sarah's preference) maps directly to "Kinky"
  • Ownable — once registered, defensible
  • Tagline-friendly — "Kinky Store — your playful intimate shop" / "Kinky Store — para los que se animan" / "Kinky Store — Asunción" all read well
  • Instagram-friendly — @kinkystore handle is likely available
  • Domain-friendly — kinkystore.com is the gold standard, kinkystore.com.py is the PY fallback

Cons

  • Word "Kinky" has variable Spanish-PY reception — some read playful, some read crude
  • Meta/Google ad filters — "Kinky" + intimate products = high rejection rate on paid ads
  • Google search — heavy competition for "kinky" in unrelated niches (kinky curls, kinky boots, kinky AI)
  • Rechtliche Position in some EU countries is restricted; not PY-relevant but matters if she ever exports
  • Possible competitor name in AR or another LATAM market — could create cross-border confusion

2. DINAPI verification — what to ask Sarah

Check Question Why it matters
Ownership ¿Tu nombre o el de tu empresa existente (la del RUC 4978694-6) es la titular? If a third party registered it, Sarah can't use it.
Class coverage ¿Qué clases tenés registradas? Need at least Class 3 (cosmetics), Class 5 (lubricants/supplements), Class 35 (retail). If only 35, must add 3+5.
Status ¿Registro vigente o en trámite? Pending ≠ live. Can't enforce until granted.
Geographic scope ¿Solo PY o también AR/UY/BO? If she ever expands, PY-only is a limitation.
Logo vs. wordmark ¿Tenés registrada solo la palabra "Kinky Store" o también un logo específico? If only wordmark, the logo design is unconstrained. If logo too, can't change visual without re-registering.

Ask Sarah to send screenshot or PDF of the DINAPI certificate (or at minimum: registration number + titular name + classes + status).

3. Ad-policy risk — what to expect

Platform "Kinky Store" + intimate products Workaround
Meta (FB + IG) — paid Likely rejected. "Adult products" + sexual connotation = restricted. Use neutral landing pages, generic brand in ads, link to the WA number. Don't use the brand in copy.
Meta (FB + IG) — organic Allowed but suppressed. Acceptable for an organic-first strategy.
Google Ads Adult products banned. "Kinky" + intimate = no go. SEO only. No paid.
TikTok Aggressive filter on "kinky" + adult. Probably not viable for paid. Organic limited.
Tigo / local PY ad networks Less strict on ad copy. Could be a viable local channel.

Bottom line: With "Kinky Store" as the brand, organic + WhatsApp is the primary acquisition channel for the first 12 months. Paid is possible but limited. This aligns with what all 5 PY competitors do anyway.
