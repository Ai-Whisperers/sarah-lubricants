Setup in Brevo

  1. Create account: https://www.brevo.com (free tier: 300 emails/day)
  2. Verify domain: enki.com.py (or use subdomain)
  3. Import contacts: From Google Sheet / IG DMs / WA opt-ins
  4. Create automation flows: Paste the email copy above into each trigger
  5. Test send: To yourself before going live
  6. Track metrics: Open rate >25%, click rate >3% (LATAM benchmarks)
