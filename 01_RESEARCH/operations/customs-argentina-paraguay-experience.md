## The 3 import paths (revisit)

### Path 1: Personal luggage (modelo旅行者)

**Limit (PY Aduana):** 1 personal effects declaration per calendar year, duty-free up to USD ~150 (verify current threshold with Aduana PY).

**Risk:** Multiple trips per year with commercial quantities = flagged as commercial = seizure + retroactive DINAVISA + fine.

### Path 2: Formal courier (DHL, FedEx)

**Limit:** Up to USD ~2,000 per shipment before formal import required.

**Process:** DHL/FedEx handles customs clearance. PY Aduana gets the package, courier pays duties, recipient reimburses.

**Pros:** Faster (3-5 days), handled by courier
**Cons:** More expensive per kg (USD 12-18/kg), still triggers DINAVISA review if cosmetics

### Path 3: Formal import via despachante

**Limit:** No limit (commercial)

**Process:** Despachante files Despacho de Importación + DINAVISA NSO + Certificado de Origen MERCOSUR + IVA + DI

**Pros:** Scalable, legal, supports B2B
**Cons:** Slow (15-30 days for DINAVISA), expensive per shipment (Gs 1.2M+)

## What Sarah should know BEFORE next modelo旅行者 trip

### Personal effects declaration rules (PY)

**Per Aduana PY (Resolution 41/2010 and updates):**
- **1 declaration per calendar year** for personal effects duty-free
- **Threshold:** USD ~150 (verify current)
- **Above threshold:** Pay duty (usually IVA + DI)
- **Commercial intent (multiple units, branded packaging):** Treated as commercial = seizure

### What "commercial intent" looks like to Aduana

| Signal | Risk |
|--------|------|
| Multiple units of same product | 🔴 High |
| Branded original packaging | 🔴 High |
| Price tags / commercial invoices | 🔴 High |
| Quantity exceeds personal use | 🟠 Medium |
| Single SKU across 20+ units | 🔴 High |
| Mixed SKUs in single category | 🟠 Medium |
| 5 units per SKU | 🟢 Low (probably personal) |
| Personal items mixed with commercial | 🟠 Confusing |

### How to mitigate modelo旅行者 risk

**Per trip:**
- ✅ Stay under 30 units per SKU (ideally under 20)
- ✅ Combine with personal travel (family visit)
- ✅ Don't repeat more than 1× per quarter
- ✅ Don't bring price lists / invoices
- ✅ Use generic packaging (not Sexitive original boxes)
- ✅ Declare honestly if asked (customs agents are trained to detect lies)
- ✅ Have personal-use narrative ready ("son para uso personal/regalo")

**Limit each quarter:**
- ✅ Max 90 units total per quarter (30 per SKU × 3 SKUs)
- ✅ Don't exceed 4 trips per year

**When to STOP using modelo旅行者:**
- ❌ After 2 trips (build pattern detection)
- ❌ If flagged or questioned
- ❌ When formal DINAVISA NSO approved (switch to formal)
- ❌ When revenue justifies formal import cost

## Formal import — what actually happens

### The paperwork chain

```
1. Sarah places order with Sexitive AR
   ↓
2. Sexitive ships via courier or maritime
   ↓
3. PY Aduana gets shipment → DESPA → 15-30 days
   ↓
4. Despachante files:
   a) Despacho de Importación (DI)
   b) DINAVISA NSO (if not already filed)
   c) Certificado de Origen MERCOSUR (saves 14-18% DI)
   d) Pago de IVA (10% on product cost)
   e) Pago de DI (0% with MERCOSUR cert, 14-18% without)
   ↓
5. SENACSA / MIC clearance (if applicable)
   ↓
6. Release to Sarah's address
```

### The realistic timeline

| Step | Time |
|------|------|
| Sexitive processes order | 3-7 days |
| Shipping (AR→PY via courier) | 3-5 days |
| Aduana clearance (via despachante) | 5-15 days |
| DINAVISA NSO (first time, per SKU) | 15-30 days |
| **Total first shipment** | **30-60 days** |
| **Subsequent shipments** | **15-25 days** |

## Common Aduana complications (and how to handle)

### Issue 1: "Producto no tiene registro DINAVISA"

**Scenario:** Aduana checks products, finds no DINAVISA NSO.

**Solution:** Despachante files DINAVISA in 15-30 days. Goods held in Aduana warehouse (storage fee per day).

**Prevention:** File DINAVISA NSO BEFORE first shipment (3-4 weeks ahead).

### Issue 2: "Certificado de Origen no coincide"

**Scenario:** Sexitive AR didn't issue MERCOSUR certificate, or it has wrong info.

**Solution:** Pay full 14-18% DI + retroactively request certificate.

**Prevention:** Confirm certificate availability with Sexitive AR before order.

### Issue 3: "Cantidad sugiere uso comercial"

**Scenario:** Aduana flags modelo旅行者 shipment as commercial (too many units).

**Solution:** Pay duty (may be retroactive DINAVISA). Worst case: seizure.

**Prevention:** Keep quantities low (<30 per SKU). Combine with personal travel.

### Issue 4: "Certificado de ANMAT falta"

**Scenario:** Aduana wants to verify product safety via ANMAT cert.

**Solution:** Despachante requests from Sexitive (can take 5-10 days).

**Prevention:** Get ANMAT certs at order placement, include with shipment.

### Issue 5: "Empaque no coincide con factura"

**Scenario:** Number of units in box doesn't match commercial invoice.

**Solution:** Clarify with despachante + Sexitive. May delay clearance.

**Prevention:** Verify packaging at Sexitive warehouse before shipment leaves.

## When to call despachante vs handle yourself

**Call despachante when:**
- ✅ Formal import (always)
- ✅ Modelo旅行者 with >20 units/SKU
- ✅ New product category (not yet registered)
- ✅ Any Aduana question (don't lie to customs)

**Handle yourself when:**
- ✅ Modelo旅行者 with <10 units/SKU
- ✅ Personal luggage size clearly personal
- ✅ You're sure of rules
