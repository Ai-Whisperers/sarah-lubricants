## 🔒 Regla de este proyecto (para Kiki / Ivan / Erebus)

**NO se crean sitios web separados para cuestionarios.** Los cuestionarios son archivos `.md` en este repositorio. Esta carpeta `/sarah/` es una **vista curada** que apunta a esos archivos.

**Para agregar un nuevo cuestionario:**
1. Crear `02_MEETINGS/2026-MM-DD-cuestionario-topic.md` (en español, fecha en nombre)
2. Copiar a `/sarah/questionnaires/NN-topic.md`
3. Actualizar el link en este `index.md`
4. Kiki sabe que hay un nuevo cuestionario activo

**Para archivar un cuestionario contestado:**
1. Mover de `/sarah/questionnaires/pending.md` a `/sarah/questionnaires/answered/`
2. Marcar como ✅ en este index
3. No borrar — el historial es valioso
